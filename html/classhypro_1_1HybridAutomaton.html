Detailed Description

template<typename Number, typename State = State_t<Number,Number>>
class hypro::HybridAutomaton< Number, State >

Class for linear hybrid automata.

Template Parameters
NumberNumber type.

The automaton owns all the locations and transitions. Newly created locations and transitions can only be added as unique pointers. If another class/object requests a location/transition, then only normal pointers are returned.
