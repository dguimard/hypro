Detailed Description

template<typename Number, typename tNumber, typename Representation, typename... Rargs>
class hypro::State< Number, tNumber, Representation, Rargs >

Class for state.

A state can be used in the classical sense, i.e. as a combination of location and state set. Furthermore we allow to to hold more than one set, aiming at providing datastructures for decoupled systems where state sets inside one state can be handled independently.

Template Parameters
NumberThe used number type.
RepresentationThe type of the primary state set.
RargsThe type of the additional state sets.
